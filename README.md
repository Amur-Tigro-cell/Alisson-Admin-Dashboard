# Admin Dashboard

This project is a static admin dashboard built with HTML and CSS, inspired by The Odin Project’s dashboard assignment.

## Features
- Responsive sidebar navigation
- Top bar with search and profile greeting
- Project cards section
- Announcements and trending panels
- Custom profile avatar and name
- Clean, modern layout and styling

## How to Use
1. Open `index.html` in your browser to view the dashboard.
2. Customize your profile picture by replacing `subrodp.jpg` in the project folder.
3. Edit project cards, announcements, or trending items directly in the HTML.

## Technologies
- HTML5
- CSS3
- Google Fonts (Roboto)
